Moose

El sistema d'objectes de Perl 5

  • Simple, bàsic, esquelètic, flexible.

  • Ofereix poca ajuda per a la programació orientada a objectes.

Moose

Moose és un sistema d'objectes complet i potent

  • Està construït al damunt del sistema d'objectes de Perl 5.

  • Pren característiques d'altres llenguatges com Smalltalk, Common Lisp i Perl 6.

  • Moose és la millor manera de programar objectes en Perl 5 modern.

Classes

En Perl 5 clàssic una classe és un paquet (o mòdul) de Perl dins un fitxer: Cat.pm

    package Cat;
    use Moose;
    
    1;

Classes

En Perl 5 modern és el mateix però amb una sintaxi alternativa que permet crear un context més específic que el del fitxer.

Això ens interessa en cas que vulguem posar més d'una classe dins el mateix fitxer.

        package Cat;
        use Moose;
    }

Conceptes avançats

  • Afavoriu la composició abans que l'herència.

  • Principi de responsabilitat única: una classe no ha de fer més feina de la que li correpon (Empleat sap d'empleats no de nòmines).

  • DRY: Don¿t Repeat Yourself.

  • Principi de substitució de Liskov: accepteu generalitzacions però sigueu específics en el que retorneu.

  • Subtipus i coercions.

  • Immutabilitat: proveu a fer els vostres objectes no modificables.
